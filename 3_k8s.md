## Задание 1. Деплоим приложение  
### Манифесты
Deployment  
```yaml
---
apiVersion: apps/v1
kind: Deployment
metadata:
  name: fixit
spec:
  replicas: 3
  selector:
    matchLabels:
      app: fixit
  template:
    metadata:
      name: fixit
      labels:
        app: fixit
    spec:
      containers:
      - image: rsafin12/iowa:v2
        name: main
```
Service  
```yaml
---
apiVersion: v1
kind: Service
metadata:
  name: fixit
spec:
  type: NodePort
  ports:
  - protocol: TCP
    port: 8080
    targetPort: 8080
    nodePort: 30800
  selector:
    app: fixit
```
